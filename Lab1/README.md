

Лабораторная 1
--------------
> Сверстать страницу в соответствии с выданным вариантом:
> вариант 2 : https://passport.yandex.ru/auth (не авторизовавшись)

**Требования по соответствию:**
 1. Первоначальный внешний вид. Чем точнее, тем лучше.
 2. Интерактивность по наведению указателя мыши: изменение цвета элемента, показ стандартных браузерных тайтлов у элементов (атрибут title), изменение указателя.
 3. Интерактивность при фокусировке курсора клавиатуры: изменение цвета текста, фона, бордеров и т. д., изменение подчеркивания.
 4. Интерактивность по вводу текста в поля ввода. Сюда также входит соответствие внешнего вида вводимого текста и использование плэйсхолдера.

**Чего пока не должно быть:**
1.	Анимации.
2.	Показ кастомных всплывающих подсказок (не браузерных).
3.	Интерактивность по клику мыши (кроме фокусировки на поле ввода).
4.	Интерактивность по нажатию клавиатуры (кроме ввода текста).
5.	Адаптивность: корректное отображение при разных размерах экрана/окна.
