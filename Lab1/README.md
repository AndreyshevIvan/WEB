Лабораторная 1
--------------
> Сверстать страницу в соответствии с выданным вариантом:
> Вариант 2 : https://passport.yandex.ru/auth (не авторизовавшись)

**Требования по соответствию:**
 - Первоначальный внешний вид. Чем точнее, тем лучше.
 - Интерактивность по наведению указателя мыши: изменение цвета элемента, показ стандартных браузерных тайтлов у элементов (атрибут title), изменение указателя.
 - Интерактивность при фокусировке курсора клавиатуры: изменение цвета текста, фона, бордеров и т. д., изменение подчеркивания.
 - Интерактивность по вводу текста в поля ввода. Сюда также входит соответствие внешнего вида вводимого текста и использование плэйсхолдера.


**Чего пока не должно быть:**
 - Анимации.
 - Показ кастомных всплывающих подсказок (не браузерных).
 - Интерактивность по клику мыши (кроме фокусировки на поле ввода).
 - Интерактивность по нажатию клавиатуры (кроме ввода текста).
 - Адаптивность: корректное отображение при разных размерах экрана/окна.
